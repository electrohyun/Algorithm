## 공부 방식

1. 문제번호, 문제종류, 문제플랫폼, 풀이방식, 문제제목과 함께 새 문제 시작을 요청합니다.
2. AI가 알고리즘 폴더에 `01_플랫폼-문제번호-풀이방식.js`와 `.md` 파일을 만듭니다.
3. 해당 파일에서 문제를 풉니다.
4. 마무리를 요청하면 AI가 오늘 daily report에 문제 md 내용을 그대로 복사합니다.

자세한 진행 규칙은 [workflow](./docs/workflow.md)를 따릅니다.
