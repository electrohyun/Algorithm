# BOJ 2606 바이러스

- 백준 사이트가 잠겨있어서 솔브는 못했지만 아마도 맞은것으로 추정
- 실제 문제를 찾았다.
- 지금은 모르는 BFS 풀 때도, 알고 있는 알고리즘 풀 때도 아님
- 자바스크립트 메서드와 더 친숙해져야 할듯

오늘의 발견:

- BFS 개념 자체보다, 자바스크립트 배열/반복문/메서드가 손에 안 익어서 구현이 막혔다.
- 다만 풀이 자체는 맞았다. 좋은 풀이법은 아니다.
- 모르는 새 알고리즘을 공부할 때가 아니고, 자바스크립트 메서드가 문제다.
- 자바스크립트 메서드 훈련을 위해 아는 알고리즘을 풀어봐도 좋겠다.

잘한 점:

- BFS가 queue + visited로 움직인다는 흐름을 이해했다.
- graph[current]가 “현재 노드와 연결된 녀석들”이다.
- 문제를 풀다가 내 방식으로 연결 그룹을 만들려는 시도를 했다.
- 막힌 지점을 정확히 발견했다.

불확실한 점:

- JS 메서드 사용이 아직 불안정하다.
- 입력을 graph 형태로 바꾸는 과정이 익숙하지 않다.
- 문제를 보고 BFS를 떠올리는 단계까지는 아직 아니다.
