## 문제 요약

- 숫자 문자열 `numbers`의 각 자리를 조각으로 쓴다.
- 조각들을 이어붙여 만들 수 있는 **모든 수**(자리 개수 1개~전부, 순서 다르면 다른 수).
- 그중 **서로 다른 소수의 개수**를 반환.
- 예: `"17"` → 만들 수 있는 수 {1, 7, 17, 71} → 소수는 {7, 17, 71} → **3**

## 오늘의 목표 (JS로 굳힐 것)

- 재귀로 **순열 나열** — "지금까지 만든 것 + 남은 조각" 들고 뻗기
- 사용 표시 → 재귀 → **사용 해제**(백트래킹)
- 중복 제거 `Set`, `√n`까지 도는 소수판별

## 문제

```js
// 재귀·백트래킹 — 소수 찾기 (42839)
// 1) 조각으로 만들 수 있는 모든 수를 재귀로 나열 → found(Set)
// 2) 그중 소수만 세기(isPrime, √n까지)
function solution(numbers) {
  const found = new Set();
  const pieces = numbers.split("");

  function finder(current, rest) {
    found.add(Number(current));       // 들어오자마자 담기 → "1","17","172" 매번
    rest.forEach((piece, i) => {
      const nextRest = rest.filter((_, idx) => idx !== i); // i번째 뺀 나머지
      finder(current + piece, nextRest);                   // 붙여서 재귀
    });
  }

  finder("", pieces);

  let count = 0;
  for (const n of found) {
    if (isPrime(n)) count++;
  }
  return count;
}

function isPrime(n) {
  if (n < 2) return false;
  for (let i = 2; i <= Math.sqrt(n); i++) {
    if (n % i === 0) return false;
  }
  return true;
}
```
