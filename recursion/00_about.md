# 재귀 · 백트래킹 (Recursion / Backtracking)

- DFS의 뿌리. 순열·조합·부분집합을 "나열"하는 재귀 뼈대를 손에 붙이는 게 목표.

- 난이도 중하 → 중상

(
하하: 배열, 문자열 기초
하: 정렬, 해시, 스택/큐 기본
중하: BFS/DFS 기본
중: BFS 최단거리, DFS 연결요소
중상: 백트래킹, 다중 BFS, 조건 많은 그래프
상: 다익스트라, DP+그래프, 트리 DP
상상: 세그트리, 고급 그래프, 어려운 DP 🤔
)

- 재귀 함수 = 스스로를 부르는 함수. **종료조건(base case)** + **한 단계 줄여 다시 호출** 두 축.
- 백트래킹 = 재귀로 경우의 수를 뻗다가, 아니면 되돌아옴(사용 표시 → 재귀 → **사용 해제**).
- 순열: 순서 O. 조합: 순서 X. 부분집합: 넣냐/빼냐.
