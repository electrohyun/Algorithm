# 스택 · 큐 (Stack / Queue)

- BFS·DFS의 재료 자료구조

- 난이도 하

(
하하: 배열, 문자열 기초
하: 정렬, 해시, 스택/큐 기본
중하: BFS/DFS 기본
중: BFS 최단거리, DFS 연결요소
중상: 백트래킹, 다중 BFS, 조건 많은 그래프
상: 다익스트라, DP+그래프, 트리 DP
상상: 세그트리, 고급 그래프, 어려운 DP 🤔
)

- 스택(Stack): LIFO(후입선출). `push`/`pop`. DFS의 재료.
- 큐(Queue): FIFO(선입선출). `push`/`shift`. BFS의 재료.
- 덱(Deque): 양쪽에서 넣고 뺄 수 있는 큐.
